# 01 — Start Here

> ⏱ **5 minutes** · 🎯 **Decide if this is for you**

## What is the Copilot Cowork Dojo?

A small framework that turns *casual Copilot users* into *trained Copilot coworkers*.

It gives you:

- A **7-step pipeline** so you stop skipping the steps that matter.
- **34 named skills** so you and your team share a vocabulary.
- **Real-world scenarios** for HR, Finance, Sales, and Project Management.
- **A prompt library** so you stop reinventing prompts from scratch.
- **A memory vault** so the team gets smarter session by session.

## Is it for me?

Tick what's true:

- [ ] I (or my team) use Microsoft 365 Copilot weekly.
- [ ] I produce knowledge work — emails, memos, recaps, briefs, status updates.
- [ ] I've shipped a Copilot output that turned out to have a mistake in it.
- [ ] My team uses Copilot inconsistently — some are great, some struggle.
- [ ] I want fewer revisions, not more clever prompts.

**3+ ticks → keep reading.**

## Is it NOT for me?

- You're a software developer training **AI agents**. Use the [Copilot Agents Dojo](https://github.com/andreaswasita/copilot-agents-dojo) instead.
- You want a tool. This is a *framework* — markdown files and a workflow. The leverage is in the practice.
- You want a magic prompt. There isn't one.

## What does "trained" actually look like?

| Untrained | Trained |
|---|---|
| "Summarize this." → accepts the result | Frames audience + altitude → verifies every fact → ships with citations |
| Same generic prompt for every task | Pulls from a 50-prompt library, adapted to the task |
| Hallucinated owner name in a recap → goes out unnoticed | Hallucination caught at the verify gate → never ships |
| Each teammate solves the same problem from scratch | Solved patterns live in `memory/patterns/` — searchable, shared |

## How long until it pays off?

| When | What you get |
|---|---|
| End of your first session (15 min) | One real deliverable shipped through the pipeline |
| End of week 1 (3 sessions) | The pipeline starts feeling automatic |
| End of month 1 (10+ sessions) | First patterns promoted; you stop making the same mistake |
| End of quarter 1 | Team-wide vocabulary; measurable drop in revision cycles |
