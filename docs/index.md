## The mandatory workflow

```
CLARIFY → PLAN → DRAFT → REVIEW → REFINE → SHIP → LEARN
```

Every non-trivial knowledge-work deliverable follows the same 7 steps. No skipping. No improvising.

## What's inside

- **24 production skills** — 6 core kata + 6 flow waza + 7 practical kumite + 3 multi-agent cowork + 2 meta dō
- **4 real-world scenarios** — HR, Finance, Sales, Project Management
- **5 personas** — analyst, project coordinator, comms lead, people manager, executive assistant
- **Prompt library** — patterns for the 80% of recurring knowledge work
- **Memory vault** — persistent, queryable knowledge graph
- **Governance** — responsible AI, data handling, escalation playbooks
- **Facilitator guide** — half-day workshop and multi-week formats
- **PowerShell scripts** — init, verify, and lesson-promotion helpers
